# Portfolio Website

Website em Flask para exibir seus projetos, habilidades e contato de forma moderna e objetiva.

## O que inclui

- landing page criativa com hero e chamadas para ação
- seção de projetos com filtros por tecnologia
- apresentação de habilidades e propostas de valor
- link direto para GitHub e contato por e-mail

## Tecnologias

- Python
- Flask
- HTML
- CSS
- JavaScript

## Como rodar

```bash
cd portfolio
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

## Personalização

- Atualize `portfolio/app.py` com seu usuário do GitHub para que os links dos projetos apontem para seus repositórios.
- O e-mail de contato já foi atualizado para `firmenetto@gmail.com`.
- Substitua as descrições dos projetos pelo que melhor representar sua experiência.
